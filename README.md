# TripX Assignment

A Next.js app where users can log in and browse a list of travel destinations.

## Stack

Next.js 16, React 19, TypeScript, Tailwind CSS, Zod for validation

## Features

- **Login** — Login page with form validation and error feedback
- **Route protection** — Middleware redirects unauthenticated users away from protected pages
- **Destinations** — Browse a list of travel destinations
- **Logout** — Users can log out and are redirected to the login page
- **Booking code** — Booking code from login is stored in localStorage and displayed on the destinations page

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
